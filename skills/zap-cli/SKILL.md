# zap-cli

Use this skill when invoking or extending the Zap CLI.

## Commands

- `zap init <dir>`
- `zap new <slug>`
- `zap validate [Zap.md]`
- `zap lint [Zap.md]`
- `zap run <Zap.md> [--json]`
- `zap doctor`
- `zap docs [topic]`
- `zap skills`

CLI runs are plan-only by default. Use `--live` only after budget and provider-key checks pass.
